Easy!Appointments - Booking Demo

Welcome to the demonstration page of Easy!Appointments. In this demo you will be able to book an appointment and experience the integrated booking wizard!

The booking wizard was built with simplicity in mind, making it extremely easy to use. Your customers will now be able to book appointments by following four simple steps till the appointment confirmation.

This demo does not contain any backend logic though which means that you will have to install Easy!Appointments in your own server in order to be able to use the numerous features and possible configuration variations.
