#Documentation

This page contains the documentation of the project for various releases. You will find useful information about installing and configuring the application, enabling the Google Calendar synchronization, troubleshooting guides and many many more.

The documentation is version-related which means that you must select the appropriate version from the list below, otherwise you might end up reading something that does not apply do your own version.
